    public addDynamicAnim(scene: Phaser.Scene, animKey: string, framePrefix: string, atlasKey: string, start: Number, end: Number, frameRate: Number, repeat: Number, repeatDelay: Number) source 

    import {addDynamicAnim} from 'src/util/Animation'

    Adds a dynamic animation to the given scene. A dynamic animation is a tradition animation where it looks like the entity is moving.

    Params:

    NameTypeAttributeDescription
    scene Phaser.Scene

    The scene to add the animation to
    animKey string

    The key that the animation will be associated with
    framePrefix string

    The prefix of the key that describes the file
    atlasKey string

    The key that describes the loaded Atlas
    start Number

    The number of the first frame
    end Number

    The number of the last frame
    frameRate Number
    • optional
    • default: 10

    The frame rate of playback in fps
    repeat Number
    • optional
    • default: -1

    Number of times to repeat the animation (-1 for infinity)
    repeatDelay Number
    • optional
    • default: 0

    Delay before the animation repeats. Value given in milliseconds.
